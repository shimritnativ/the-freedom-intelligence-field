↗ How to Find US / UK / Canada Buyers in GHL Directly

Phone country code data lives in GHL, not in our admin dashboard. To catch ALL members (ads + organic) from these regions, filter directly in GHL every morning.

1

Open GHL Contacts

Go to the Contacts view in GoHighLevel.

2

Apply these filters

  • Phone starts with +1 (USA / Canada) or +44 (UK)
  • Tag contains Power Reset buyer (or your equivalent purchase tag)
  • Date Added in last 30 days
3

Sort by Date Added descending

Newest members first. Work down the list, oldest still-relevant last.

4

For each member on the list

  • Check the day since purchase → determines which message they should have received (see Quick Lookup above)
  • Check GHL activity log → did the automation actually deliver?
  • If undelivered → send that message manually via WhatsApp Business
  • Add or update Notes + Next Follow-Up Date in their GHL record
5

Save this as a Smart List

In GHL, save the filter as a Smart List called "US/CAN/UK Manual Outreach" so you can open it in one click every morning instead of rebuilding the filter.

▲
Why manual instead of automated: GHL's WhatsApp automation is unreliable for numbers with +1 or +44 country codes. Meta/GHL cannot reliably deliver template messages to these regions. This is a known limitation, not a bug you can fix. Manual WhatsApp Business is the workaround.
